Shortest path algortihms learning environment

The main purpose of this app is assisting anyone interested in learning about shortest path algortihms.

Shortest path algorithms are designed to solve the shortest path problem. The shortest path problem is commonly defined for weighted graphs. These graphs are characterized by the fact that each edge in the graph is assigned some number, called a weight. In the context of the shortest path problem, weights are interpreted as lengths of edges. The solution to the problem is such a sequence of edges connecting two chosen nodes (or a sequence of nodes connected by edges, depending on interpretation), that the sum of edges lengths is minimal.
